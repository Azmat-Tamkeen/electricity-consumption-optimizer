# Voltwise-Electricity Consumption and Optimizer
A smart web application to track electricity usage,analyze consumption,predict monthly bills,and optimize energy costs usinng Flask,OCR,and analytics.
## Features
-User Login and Registration
-Dashboard
-Manual Meter Reading
-Bill Prediction
-Appliance Tracking
-Budget Alerts
-Reports and Analytics
## Technologies
Python,Flask,HTML,CSS,JavaScript,SQLite,OCR
